# Classifieds API

Created with GraphQL, Node.js and MongoDB.  

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Environment variables

Add .env file to the project:

```bash
MONGO_URL=<VALUE>
JWT_SECRET=<VALUE>
PORT=<VALUE>
```
